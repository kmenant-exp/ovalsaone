## Technologies Utilisées
- **Eleventy (11ty)** : Générateur de site statique moderne
- **Templates Liquid** : Moteur de template pour les pages
- **Nunjucks** : Moteur de template pour les layouts
- **Cloudflare Pages** : Hébergement et Pages Functions (TypeScript)
- **Cloudflare D1** : Base de données SQLite distribuée
- **Resend** : Envoi d'emails transactionnels
- **Cloudflare Turnstile** : Protection anti-bot des formulaires
- **Decap CMS** : Gestion de contenu via GitHub
